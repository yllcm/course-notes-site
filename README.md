# 大学课程笔记

清华大学部分核心课程笔记，使用 [Quartz](https://quartz.jzhao.xyz/) 构建发布。

## 课程

- **数学分析** — 一元与多元微积分、级数论、Riemann/Lebesgue 积分、隐函数定理
- **离散数学** — 数理逻辑、集合论、图论、代数结构（群）
- **大学物理（英）** — 力学、刚体、流体、狭义相对论、振动与波、热学

## 本地开发

```bash
npm install
npx quartz plugin install --from-config
npx quartz build --serve
```

访问 http://localhost:8080
