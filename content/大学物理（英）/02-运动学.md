## 一维运动 (1D Motion)

基础物理量与微积分关系：

- **位移**：$\Delta x$

- **平均速度**：$\bar{v} = \frac{\Delta x}{\Delta t}$；**瞬时速度**：$v = \frac{dx}{dt}$

- **平均加速度**：$\bar{a} = \frac{\Delta v}{\Delta t}$；**瞬时加速度**：$a = \frac{dv}{dt} = \frac{d^2x}{dt^2}$

- **积分关系**：

- $v - v_0 = \int_{t_0}^{t} a(t) dt$
	
- $x - x_0 = \int_{t_0}^{t} v(t) dt$
	
- **恒等式推导**：利用 $v \frac{dv}{dx} = a$，可得 $\int_{v_0}^{v} v dv = \int_{x_0}^{x} a dx$。当 $a$ 为常数时，即为初中物理经典的 $v^2 - v_0^2 = 2a(x - x_0)$。

## 二维与三维运动 (2D or 3D Motion)

使用向量描述空间位置：

- **位置向量**：$\vec{r} = x\mathbf{i} + y\mathbf{j} + z\mathbf{k}$

- **位移向量**：$\Delta \vec{r} = \vec{r}_2 - \vec{r}_1$

- **速度向量**：$\vec{v} = \frac{d\vec{r}}{dt} = \frac{dx}{dt}\mathbf{i} + \frac{dy}{dt}\mathbf{j} + \frac{dz}{dt}\mathbf{k}$

## 圆周运动 (Circular Motion)

- **角位移与角速度**：$\theta = \omega t$；$\omega = \frac{d\theta}{dt}$（Rad/s）

- **位置描述**：$\vec{r} = R\cos(\omega t)\mathbf{i} + R\sin(\omega t)\mathbf{j}$

- **速度**：$\vec{v} = \frac{d\vec{r}}{dt}$，其模长 $|\vec{v}| = \omega R$

- **加速度**：$\vec{a} = -\omega^2 R\cos(\omega t)\mathbf{i} - \omega^2 R\sin(\omega t)\mathbf{j} = -\omega^2 \vec{r}$

- _注：这表明匀速圆周运动的加速度始终指向圆心（向心加速度）。_

## 极坐标系下的运动 (Polar Coordinates)


### 单位向量及其导数

- $\vec{e}_r = \cos\theta \mathbf{i} + \sin\theta \mathbf{j}$

- $\vec{e}_\theta = -\sin\theta \mathbf{i} + \cos\theta \mathbf{j}$

- **关键导数**（注意是对 $\theta$ 求导）
$$
\frac{d\vec{e}_r}{d\theta} = \vec{e}_\theta,\quad\frac{d\vec{e}_\theta}{d\theta} = -\vec{e}_r
$$

### 极坐标下的速度与加速度公式

- **速度 $\vec{v}$**：根据[[01-向量和标量#向量微积分 Vector Derivatives|向量微积分]]部分给出的公式

$$
\begin{align*}\frac{d}{dt}\vec{r}(t) 
&= \frac{dr(t)}{dt}\vec{e}_r + r(t)\frac{d\vec{e}_r}{dt}\\
&=\frac{dr}{dt}\vec{e}_r+r\frac{d\theta}{dt}\frac{d\vec{e}_r}{d\theta}\\
&=\frac{dr}{dt}\vec{e}_r+r\frac{d\theta}{dt}\frac{d\vec{e}_r}{d\theta}\\
&=\frac{dr}{dt}\vec{e}_r+r\frac{d\theta}{dt} \vec{e}_\theta\end{align*}
$$

(包含径向速度和横向速度 $v_\theta = wr$)

- **加速度 $\vec{a}$**：

$$
\vec{a} = \left( \frac{d^2r}{dt^2} - r\left(\frac{d\theta}{dt}\right)^2 \right)\vec{e}_r + \left( r\frac{d^2\theta}{dt^2} + 2\frac{d\theta}{dt}\frac{dr}{dt} \right)\vec{e}_\theta
$$

- $\frac{d^2r}{dt^2} - r\omega^2$：径向加速度（含向心项）。
	
- $r\alpha + 2\omega\dot{r}$：横向加速度（含**科里奥利加速度**项）。
